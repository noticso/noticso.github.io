# OverTheWire Bandit Walkthroughs

This section contains my notes and walkthroughs for the **Bandit** wargame.

Bandit is a beginner-friendly Linux and security learning environment. The early levels focus on navigating files, working with permissions, understanding encodings, using SSH, and interacting with local network services.

!!! warning "Spoilers"
    These pages contain complete walkthroughs and passwords captured during my own progress. Passwords on the official platform can change over time.

## Format used in every page

Each walkthrough follows the same structure:

1. **Objective** — what the level asks for
2. **Solution** — the reasoning and commands used
3. **Password Retrieved** — the result from my run
4. **Key Takeaway** — the reusable Linux or networking concept
