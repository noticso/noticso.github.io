# About This Site

This site is a personal cybersecurity learning journal maintained by **Giovanni D'Agostino**.

The purpose is to document practical progress, improve technical writing in English, and build a public portfolio of authorized learning activities.

## Scope

The content focuses on:

- Linux fundamentals
- Basic networking
- Command-line tools
- Authorized wargames and labs
- Notes on problem-solving methodology

## Responsible use

All examples are intended for environments where testing is explicitly authorized. Never use commands, scans, credentials, or techniques against systems you do not own or do not have permission to test.
